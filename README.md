How to use it :
* clone this repo
* Install python
* Install pip ( package manager for python language )
* Install virtualenv
* create virtualenv folder and activate the env
  ```bash
  $ virtualenv env
  $ source /env/bin/activate
  ```
* Install the requirement.txt
  ```bash 
  $ pip install -r requirement.txt
  ```
